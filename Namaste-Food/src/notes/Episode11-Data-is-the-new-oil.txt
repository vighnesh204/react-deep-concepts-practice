🚀 Higher Order components
    => Higher Order component is function that takes a component and returns a component
    => Higher order components basically takes a component as an input then it enhances that component, add some features to that component and returns it back, it acts like a enhancer kind of function that takes the existing component and just enhances it, modifies that component and returns it back
    => Higher Order components are pure functions, what do you mean by pure functions ?
    => i am passing RestaurantCard inside this function / Higher order component 
    => i will not modify the code of RestaurantCard
    => i will not change the behaviour of RestaurantCard
    => i am just taking RestaurantCard as an input and i am using it exactly how we use RestaurantCard
    => i am just enhancing it
